Project Details
Frontend
The frontend is built with React and Vite.
The main entry point is main.jsx.
The main application component is App.jsx.
The application uses React Router for routing between different pages.
Backend
The backend is built with Express and interacts with the OpenAI API.
The main server file is server.js.
The server listens for POST requests at /api/prompt and forwards the prompt to the OpenAI API.
Components
Welcome: The welcome page with a link to the prompt page.
Prompt: The page where users can enter their prompt and submit it.
Result: The page that displays the result from the OpenAI API.
Configuration
The Vite configuration is in vite.config.js.
ESLint configuration is in eslint.config.js.
